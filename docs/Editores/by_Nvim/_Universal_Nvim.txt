Universal VIM

Instalacao: VIM:: `sudo apt install vim`

Utils:
 - se for fazer comando Maiusculo é só usar shif+Letra, não precisa primeiro deuxa-la maiuscula.

O Editor Possui 4 modos: [
Comandos: vc sempre volta pra esse ao dar Esc, acesso: Esc, 

Inserção : vc escreve coisas novas , acesso: i,

Substituição: replace vc escreve por cima , acesso: 2x tecla INSERT,

Visual: , acesso: Esc v,
]

Modo/Insercao

Atalhos : 

desfazer : u, significa: undo
refazer: ctr+r ,significa: redo
tab_na_linha : ctr+t da o tab, desfazer: ctr+d
repete_linha_de_cima: ctr+y
repete_linha_de_baixo: ctr+e
sair_sem_salvar: q, significa quit
sair_sem_salvar_forcando: q!, significa: o ! é forçar
salvar_e_continuar: w, significa: write / escrever
salvar_e_sair: wq!, significa: write quit firme, salvar sair forçado.
salvar_com_outro_nome: w ./localidade/nomearquivoNovo.extensao

mover_entre_palavras/ou_linhas: shift+SETAS

---

Modo Visual : acesso: Esc v

copiar_selecionado: seleciona y, 
copia_linha: yy, 
colar: p, significa: paste / colar
deletar: d, significa: delete
selecionar_varias_linhas: seta pra cima ou pra baixo.
recortar_jogarPraMemoria: td, dps ao colar pode colar quantas vezes quiser porque foi pra memória.

usar_atalhos_com_numerosLinha: número De Linhas <comandoAtalho> ex: 9 dd vai apagar da linha atual até 9 linhas pra baixo,para esses é bom usar o numerador padrão de linhas do vim.

unir_fim_da/linhaDeBaixoVomADeCima. shif+j
