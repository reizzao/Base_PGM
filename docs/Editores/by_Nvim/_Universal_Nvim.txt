# Universal Neovim

PrimeiroPasso:
- primeiro tem que instalar o neovim na maquina : `sudo apt install neovim`
- depois instala as novas versoes modificadas.

Comandos:
  Editor
    - abrir_o_editor : `nvim`

Configuracoes:
 Deixar_Cursor_Airb : `vim.cmd([[ set gui cursor= ]])`

---
Significados: 

- Buffer: buffet é a memoria, algo/arquivo que ainda não existe enquanto não for salvo, depois que é salvo sai da memória auxiliar e vira algo/arquivo concreto

Universal VIM

Instalacao: VIM:: `sudo apt install vim`

O Editor Possui 3 modos: [
Comandos: vc sempre volta pra esse ao dar Esc, acesso: Esc, 

Inserção : vc escreve coisas novas , acesso: i,

Substituição: replace vc escreve por cima , acesso: 2x tecla INSERT,
]

Modo/Insercao

Atalhos : 

desfazer : u, significa: undo
refazer: ctr+r ,significa: redo
tab_na_linha : ctr+t da o tab, desfazer: ctr+d
repete_linha_de_cima: ctr+y
repete_linha_de_baixo: ctr+e
